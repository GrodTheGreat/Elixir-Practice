# Elixir Practice
This repo is a place to store some projects as I practice Elixir

## Projects
### Project 1: Hello World
**Overview:** Start with the classic "Hello World" program to get familiar with the syntax and basic structure of an Elixir program.

**Skills to Focus On:**
- Understanding basic syntax and structure of Elixir code.
- Running Elixir scripts and programs.

### Project 2: Simple Calculator
**Overview:** Build a simple command-line calculator that can perform basic arithmetic operations like addition, subtraction, multiplication, and division.

**Skills to Focus On:**
- Working with variables and data types.
- Basic arithmetic operations.
- Handling user input.

### Project 3: To-Do List
**Overview:** Create a simple to-do list application where users can add, edit, delete, and mark tasks as completed.

**Skills to Focus On:**
- Defining and structuring modules.
- Managing state.
- Implementing basic CRUD operations.

### Project 4: URL Shortener
**Overview:** Build a URL shortener service that takes a long URL as input and generates a short, unique identifier for it.

**Skills to Focus On:**
- Working with Elixir's built-in `GenServer` for state management.
- Handling HTTP requests (consider using Plug for web server interactions).

### Project 5: Chat Application
**Overview:** Develop a simple chat application that allows multiple users to join chat rooms and exchange messages in real-time.

**Skills to Focus On:**
- Implementing real-time communication using WebSockets (Phoenix Channels can be useful).
- Managing concurrent connections and state.

### Project 6: RESTful API
**Overview:** Create a RESTful API for managing resources (e.g., users, posts) using Elixir and Phoenix framework.

**Skills to Focus On:**
- Understanding REST principles.
- Handling HTTP requests and responses.
- Implementing CRUD operations for resources.

### Project 7: Authentication System
**Overview:** Implement user authentication and authorization system with features like sign-up, login, logout, and access control.

**Skills to Focus On:**
- Working with authentication libraries like Guardian.
- Implementing authentication and authorization middleware.

### Project 8: File Uploader
**Overview:** Develop a file uploading service that allows users to upload files, manage them, and access them later.

**Skills to Focus On:**
- Handling file uploads.
- Storing files on disk or cloud storage (consider using libraries like Arc or ExAWS).
- Implementing file management features.

### Project 9: Task Scheduler
**Overview:** Build a task scheduler service that allows users to schedule tasks to be executed at specific times.

**Skills to Focus On:**
- Working with Elixir's built-in scheduler or using external libraries like Quantum.
- Understanding concurrency and parallelism in task execution.

### Project 10: E-commerce Platform
**Overview:** Develop a basic e-commerce platform with features like product listing, shopping cart, checkout, and order management.

**Skills to Focus On:**
- Designing and structuring a complex application.
- Implementing business logic for e-commerce features.
- Integrating with payment gateways (if desired).
